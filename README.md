# api-baraja
Este proyecto es una aplicación WebGL desarrollada en Unity que consume una API REST falsa alojada en GitHub para obtener los IDs de las cartas asociadas a un jugador. A partir de esos IDs, el sistema consulta una API de terceros para obtener la información detallada de cada carta (nombre, especie e imagen) y la muestra en una interfaz gráfica interactiva.

El usuario ingresa su ID, visualiza su baraja personalizada y puede cambiar de jugador en cualquier momento. El proyecto está desplegado en GitHub Pages y utiliza UnityWebRequest para realizar las peticiones HTTP.
